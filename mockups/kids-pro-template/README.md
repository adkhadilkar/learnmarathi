# Kids Pro Website Mockup

Static mockup for a more professional and kid-friendly Marathi Shikuya interface.

## Files

- `index.html` - reviewable single-page website mockup
- `styles.css` - responsive visual system, animations, mascot, scenes, rewards
- `script.js` - interactions, audio fallback, easter eggs, rewards

## Easter Eggs

- Tap the acorn logo.
- Poke Shekru.
- Tap the star bank.
- Tap `मोदक` in the footer.
- Type `SHIKUYA`.

## Notes

This folder is intentionally separate from the current React app. It can be used as a design reference before porting the confirmed direction into the real app.

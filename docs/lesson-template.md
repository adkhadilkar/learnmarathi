# Lesson Template

Each lesson follows a 3–7 minute arc with five parts:

1. **Warm-up (20–30s):** quick recognition (tap the right अक्षर)
2. **Teach (60–90s):** show letter/मात्रा with clear audio + example
3. **Practice (2–4 min):** 2–3 mini-games (varied)
4. **Apply (60–90s):** read a word/sentence in context or write it
5. **Reward (10–20s):** badge/coin/story unlock

## Lesson data contract
Lesson data lives in `content/lessons/**` and follows `content/schema/lesson.schema.json`.

Required fields:
- `id`, `level`, `title`
- `targetSkills`
- `warmup`, `teach`, `practice`, `apply`, `reward`
- `assets`

This template ensures consistency across letter recognition, handwriting, barakhadi mastery, and reading comprehension.

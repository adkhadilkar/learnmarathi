# MVP Scope

## MVP goal
Prove retention and measurable literacy gains within 6–10 weeks.

## Included features
- Onboarding + placement test.
- Level 1 recognition lessons (स्वर/व्यंजन).
- Basic tracing (guided + freestyle).
- Level 2 starter barाखडी (8–10 combos).
- 30–50 words, 50–100 sentences (controlled vocabulary).
- 10 short stories (graded).
- Streak + simple rewards.
- Parent dashboard (basic progress).

## Excluded (later)
- Speech scoring.
- Leagues.
- Advanced grammar modules.
- Large story library.
